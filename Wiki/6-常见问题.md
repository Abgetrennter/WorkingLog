# 6. 常见问题解决方案 (FAQ)

## Q1: 保存时提示 "文件被占用" 或 "IOException"？
**A**: 请检查生成的 Excel 文件（`Data/worklog_yyyyMM.xlsx`）是否正在被 Microsoft Excel 或 WPS 打开。
- **解决方法**: 关闭 Excel 软件，然后再次点击保存。

## Q2: 为什么导入的数据没有显示？
**A**: 
1. 检查导入向导中预览列表是否为空。
2. 确认导入的数据日期是否属于当前选择的月份。系统会自动将数据归档到对应月份的文件中，您可能需要切换月份查看。
3. 检查 Excel 表头是否包含中文 "日期"、"标题" 等关键列。

## Q3: 模板修改后没有生效？
**A**: 
- 在 "分类管理" 界面修改后，必须点击工具栏的 **"保存"** 按钮，配置才会写入磁盘。
- 如果仍未生效，尝试点击主界面的 **"刷新"** 按钮或重启程序。

## Q4: 每日总结自动生成的文字不准确？
**A**: 
- 每日总结默认逻辑是提取当日所有事项的 `标题` 并用分号连接。
- 这只是一个草稿，您可以在弹出的对话框中手动编辑、润色，然后再保存。

## Q5: 如何迁移到新电脑？
**A**: 
- 将程序目录下的 `Data` 文件夹（日志数据）和 `Templates` 文件夹（模板配置）复制到新电脑的程序目录下即可。

## Q6: 支持 Mac 或 Linux 吗？
**A**: 
- 目前仅支持 Windows (WinForms 技术栈)。
- 未来可能通过迁移到 MAUI 或 Avalonia 实现跨平台，但目前暂无计划。

## Q7: 日志文件损坏怎么办？
**A**: 
- 系统在写入 Excel 前没有自动备份旧文件（后续版本会改进）。
- 建议您定期手动备份 `Data` 目录。
- 如果文件无法打开，尝试用 Excel 的 "打开并修复" 功能。
